# Project State

## Mission

Desktop Cat turns a real cat into a lightweight macOS desktop companion. The product should feel calm, personal, and alive without becoming a game, chatbot, or notification tool.

## Current Version

Internal Alpha V1.

## Current Sprint

Sprint 1: build the first runnable macOS prototype.

Sprint 1 target:

> A small demo black cat can appear on the macOS desktop.

## V1 Scope

Included in V1:

- macOS desktop pet
- demo black cat
- unified cartoon visual style
- basic motion library
- simple behavior brain
- drag interaction
- petting interaction
- feeding interaction
- simple toy interaction
- sitting and hydration reminders
- AI-generated user cat using the founder's API during internal alpha

Excluded from V1:

- growth system
- long-term memory
- multi-cat support
- cloud sync
- commercialization
- complex personality learning
- full chatbot behavior

## Workflow

Founder describes needs in natural language.

Product Architect converts those needs into scope, decisions, and Codex-ready tasks.

Codex implements the tasks, keeps changes small, and updates documentation when decisions change.

## Current Rule

Keep the prototype small and runnable. Do not add AI, uploads, reminders, growth, or cloud services to Sprint 1.
