# Master Plan

## Guiding Principle

Every document must help Codex make the next correct decision.

## Product Goal

Build a lightweight macOS desktop companion cat that users want to keep open while working.

The product should feel like quiet companionship, not a game, chatbot, or showcase of technical tricks.

## Alpha V1 Goal

The user can download, install, open, and see a small black cartoon cat living on the desktop.

## Milestones

### Milestone 1: Project Foundation

Create the repository structure and documents that future Codex sessions can understand quickly.

Status: in progress.

### Milestone 2: First Prototype

Create a minimal macOS desktop app with a transparent window and a visible demo black cat.

Status: not started.

### Milestone 3: Basic Behavior

Add idle, walk, sit, and sleep behavior using a simple behavior brain.

Status: not started.

### Milestone 4: Basic Interaction

Add drag, petting, feeding, and simple toy interactions.

Status: not started.

### Milestone 5: AI Cat Creation

Allow users to upload cat photos and generate a stylized desktop cat through a replaceable AI service provider.

Status: not started.

### Milestone 6: macOS Distribution

Build and package a DMG suitable for internal testers.

Status: not started.

## Definition Of Done For V1

A non-technical macOS user can install the app, open it, see a personalized or demo cat on the desktop, and complete at least one simple interaction such as petting or feeding.
