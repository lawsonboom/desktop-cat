# Roadmap

## Internal Alpha

Goal: make the first black cat demo run on macOS.

Planned:

- project scaffold
- transparent desktop window
- demo black cat asset
- idle behavior
- random walking
- clean close behavior
- early DMG packaging

## V1

Goal: make a small desktop cat feel present and useful without becoming complex.

Planned:

- basic behavior brain
- basic motion library
- drag interaction
- petting interaction
- feeding interaction
- simple toy interaction
- hydration reminder
- sitting reminder
- optional launch at login setting
- speech bubbles used sparingly
- AI generation through a replaceable service layer

## V2

Candidate ideas, not part of V1:

- Life Memory from user-uploaded photos
- personal motion library
- richer cat habits
- more toys
- more desktop world behaviors

## V3

Candidate ideas, not part of V1:

- personality learning
- growth system
- multi-cat support
- cloud sync
- community sharing
- commercial plans
